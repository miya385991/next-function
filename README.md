# Next.js環境説明

## 準備
1. next.js の環境
   1. create-next-appで環境を用意してください
2. ダミーデータ 
   1. [{json}placeholderのAPIを使用](https://jsonplaceholder.typicode.com/)

まず、初めに `env`もしくは`env.local`ファイル作成します。

env.localに`NEXT_PUBLIC_API_URL= API_URL記述`を記述してください
